6. ZigZag Conversion #Medium

题目描述

The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

P   A   H   N
A P L S I I G
Y   I   R

And then read line by line: "PAHNAPLSIIGYIR"

Write the code that will take a string and make this conversion given a number of rows:

string convert(string s, int numRows);

Example 1:

Input: s = "PAYPALISHIRING", numRows = 3
Output: "PAHNAPLSIIGYIR"

Example 2:

Input: s = "PAYPALISHIRING", numRows = 4
Output: "PINALSIGYAHRPI"
Explanation:

P     I    N
A   L S  I G
Y A   H R
P     I

题目解析

首先，当字符串s的长度小于2或者numRows等于1时，直接返回s即可；其次，声明数组svec，其元素为重排后每一行的字符串，元素数量与numRows相等；然后，每2*numRows-2个字符成为一组，那么下标为i的字符在每一组中的下标为t = i % (2*numRows-2)，如果下标t小于numRows，则将其添加到svec中下标为t的字符串末尾，否则将其添加到svec中下标为2*numRows-2-t的字符串末尾；最后，将svec中的所有字符串拼接起来即可。

C++ 实现

class Solution {
public:
    string convert(string s, int numRows) {
        if (s.size() < 2 || numRows == 1) return s;
        vector<string> svec(numRows);
        for (int i = 0; i < s.size(); ++i) {
            int t = i % (2 * numRows - 2);
            if (t < numRows)
                svec[t].push_back(s[i]);
            else
                svec[2*numRows-2-t].push_back(s[i]);
        }
        string res = "";
        for (auto s : svec)
            res += s;
        return res;
    }
};
