iPhone 用户在最近看到过最多的画面是什么？

恐怕是 Face ID 识别失败的提示了。

因为要戴口罩的原因，所以很多 iPhone 用户最近都无法顺利地通过 Face ID 进行解锁。

那么，如果把挡住的鼻子和嘴巴都印在口罩上呢？

不，不是说这个。

▲图片来自：@新浪手机

是这个。

近日，产品设计师 Danielle Baskin ，也「打起了卖口罩的主意」，想打造出一个「能够让 Face ID 认出你」的口罩。

简单来说，Danielle Baskin 想做的就是将你脸被挡住的部分打印在 N95 口罩上。这样即便戴起口罩，从外面看你的脸依旧是完整的。

Danielle Baskin 还十分认真地为这款新品做了一个预约网站，根据网站上的指引，要获得这样一个口罩并不难。

第一步，你需要上传一张自拍照片（网站推荐在白天自然采光环境好的情况下拍摄）。

在收到照片以后，网站就会开始制作口罩预览，并且根据你的脸型逐点去修复对齐问题。

在收到预览并确认之后，就是最后一步，付款。这样一个个性化定制口罩并不便宜，单个零售价为 40 美元。这个价格即便你一周一换，也可以说是相当昂贵了。

官方表示，口罩卖这么贵，除了因为是个性化定制之外，也因为它的质量确实是杠杠的。

首先，这个口罩完全符合 N95 的防护标准，能够有效地隔绝空气中的飞沫。

其次，打印所使用的染料为天然染料，无毒无味且不影响透气性。

最后就是「keep it real」。通过网站表示他们可不是像那些山寨厂一样，随便把照片印上去就完事儿的。

而是通过智能算法，根据你的面部特征，点对点地进行校正，让口罩打印出来的效果尽可能地符合真实情况，不会让你看起来变胖，当然你也别想通过这个口罩来达到瘦脸的效果。

甚至为了尽可能地真实，口罩的绑带还会根据你的肤色进行上色，不给 Face ID 留一点机会。

因为 Face ID 其实是通过红外点阵投影进行三维建模的，所以不少网友质疑戴上面罩是不是真的能提高解锁率。对此，Danielle Baskin 也在 Twitter 上坦承认证成功率目前还在测试当中。

潜台词自然就是：其实我也不知道这玩意儿能不能成。

即便你财大气粗，40 美元花起来跟 4 块钱似的，也不介意戴上去之后 Face ID 到底能不能认出你，就是想买这么一个在街上回头率 100% 的口罩也没那么容易。

因为目前这个产品还在预约阶段，什么时候推出也还没确定。Danielle Baskin 表示，这个 Face ID 口罩进行投产的话，至少还要等到疫情结束之后，毕竟他们并不想在这个口罩紧缺的时候，给大伙儿添乱。

另外，为了表明自己没有囤积居奇，她还在 Twitter 上声明，她目前所有用作试验的口罩都是因为自己要经常做喷涂工作所以在很早之前囤下的。

当然，有钱有闲的人还是不少的，Danielle Baskin 透露目前在网站预约的用户已经超过了 100 位。

才 100 人？有人可能会对此作出嘲讽。但 Danielle Baskin 自己一点儿也不在乎。

Danielle Baskin 表示自己并不太介意这款产品最终能不能起到提高 Face ID 识别率的效果，也不介意这款产品的最终预约数量。她之所以推出这款产品，其实是想改变大多数人对口罩的印象：

很多人觉得走进医院，看到戴着口罩的医生就会感到莫名的恐惧和不安。但如果每个口罩都有一个特别的印记，那么氛围就会变得更加轻松和温暖。

与其在家数脚毛，不如「调教」Face ID

疫情发生后，前魅族科技高级副总裁李楠发表微博称：

Face ID 最近在中国真心变成了一个鸡肋功能。

而为这事儿操碎了心的，除了 Danielle Baskin 之外，还有广大中国网友：

反正不能出门，与其数毛，不如对 iPhone 调教一番。

实际上，Face ID 之所以被称作最好用的面部解锁，除了安全性高（可防止用照片解锁、闭眼解锁）、能够在黑夜解锁之外，还有一点就是具备动态学习功能。

在发布会上，苹果就表示，不需要多次录入，在你平常使用的过程中，系统就会自动学习你化妆/素颜、留胡子/不留胡子、戴眼镜/不戴眼镜等的样子。

唯独面对两样东西，Face ID 是没辙的，一是区别开双胞胎，二是戴口罩等情况下的大面积面部遮挡。

但随后有网友发现，其实多试几次，还是能教会 Face ID 认出你戴口罩时的样子的。

具体方法就是戴着口罩，在 Face ID 识别失败后通过输入密码解锁手机，然后不断重复上面的步骤，以此来让 Face ID 记住，它面前的这个人虽然戴着口罩，但确实就是它的主人。

不过，这个方法也并非所有用户都能试成功。

数码视频自媒体大 V @老七就表示，在日常使用中，Face ID 已经适应了他戴口罩时候的脸，并且发视频演示了戴着口罩成功解锁 Face ID。

▲图片来自：新浪微博 @老七

腾讯科技的钟文泽甚至还演示了 Face ID 无论是医用外科口罩还是 N95 口罩都能识别成功了。

▲图片来自：bilibili 钟文泽

而有不少网友则表示，即便经过数十次甚至是上百次的「调教」，Face ID 依旧不能认出自己的主人。
