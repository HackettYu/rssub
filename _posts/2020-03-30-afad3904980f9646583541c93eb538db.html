3 月初，微软公布了基于 Fluent Design 的新开始菜单设计，新设计除了一套风格统一的新图标外，去掉了代表 Metro 特色的「动态磁贴」。拥有 10 年历史的「动态磁贴」真的已经被微软抛弃了么？其实一切早有预兆。

去年 10 月，微软紧跟年初的折叠屏潮流，发布了一台双屏的笔记本电脑—— Surface Neo 。这台电脑 B 面和 C 面都是触摸屏，为了让双屏的生产力得到充足的释放。微软为它提供了一套专用的操作系统—— Windows 10X。和当年微软为 Surface 适配 Win 8 一样，Windows 10X 代表了微软对未来操作系统的设计的思考。

▲Surface Neo 图片来自：The Verge

在 Windows 10X 上，微软去掉了开始菜单的「动态磁贴」设计，要知道这是一款为触摸屏优化的系统，而当年「动态磁贴」也是面向触摸屏设计的功能，这意味着微软已经放弃了当年的设计思路。

▲Windows 10X 没有动态磁贴

果然，半年后的 Windows 10 更新开发者版本中，微软同样移除了「动态磁贴」设计。虽然 Windows Insider 团队的高级程序经理 Brandon LeBlanc 说：「动态磁贴功能并没有被移除，那些喜欢它的人依旧可以使用动态磁贴。」

但我们都知道，微软已经「放弃」了动态磁贴设计。或许下个版本，它就会从我们的视野中消失。

动态磁贴——图标与通知合二为一

为什么微软「放弃」了动态磁贴，我们要从 Windows Phone 7 时代说起。

▲ Windows Phone 7 

 

2010 年，微软为了迎合触摸屏设备的发展势头，推出了为触摸屏优化的 Windows Phone 7 系统，当时微软推出了 Metro 设计理念，将小图标改成了更方便触摸的方块设计，并创造性的将通知信息与图标合二为一。

微软的想法是，方块的 Metro 设计改善了触摸体验，方块内部的空间也可以拿来显示通知或者预览，让用户不用点击就获取应用中的消息，将原本封闭的消息通知放在桌面上，提高系统的生产力。

▲Windows Phone 界面

微软的设想十分美好，Metro 设计也带来的更陡的学习曲线，当时用户已经习惯了 iOS 和 Android 的图标设计，很难再接受这种类似瓷砖的色块设计。最终，微软凭借独特的 UI 设计收获了一批粉丝，也因为 UI 学习成本高，入场晚，没有成为主流操作系统。

▲ 为触屏优化

后来，微软将这一设计带到 PC 平台上，带来了更多的问题。首先 Metro 的与 Windows 7 的开始菜单设计几乎完全不同，对 Windows 的老用户来说，Windows 8 不亚于一个全新的系统。大量的用户无法摸清动态磁贴的使用方法，甚至完全没有利用过这个功能。

其次，动态磁贴是「不可交互」的。也就是说，它相当于一个预览板，只能按照顺序将信息在 Metro 色块中轮流展示，由于磁贴大小限制，一些信息展现的也不完整。

▲ 开始菜单进化史 图片来自：The Verge

打个比方，比如你想知道自己有哪些新邮件，你必须等邮件磁贴慢慢刷新，这本身就是一件效率低下的办法。就算你看到了你想看的邮件，你也必须点开磁贴，进入应用中查看，如果是这样，干嘛不一开始就点开程序，用列表查看邮件呢？

除此以外，当时的 Windows 8 系统还没有通知中心，应用的信息分散在每一个不同的动态磁贴上，查找起来并不方便。

▲ Win 10 通知中心

当然，微软自己也意识到了这些问题，并在 Windos 8.1 中做了一些改进措施，比如回归开始菜单，新增通知中心。但针对动态磁贴的交互性，并没有明显改进。

不过动态磁贴设计也不是一无是处，在天气、日历、日程、歌词显示等常驻信息上，动态磁贴相比静态图标有着更好的信息展示效果。而且动态磁贴提供三种尺寸和 20 多种颜色，用户可以自由的在桌面上对其进行组合，为玩家提供了个性化的选择。这些曾经是软粉喜欢动态磁贴的一大重要理由。

▲ Windows Phone 桌面  图片来自：知乎

可惜因为生态发展的原因，Windows Phone 逐渐走向衰落，用户的减少导致让相当一部分开发者放弃了对 Windows Phone 的更新和适配，也没人愿意费工夫为 动态磁贴设计做优化。大量 UWM 软件都没有适配动态磁贴功能，Metro 也就成了「丑陋的纯色大方块图标」。

微软的野望——交互式动态磁贴

为了让动态磁贴的能发挥更多的作用，微软官方也不是没有做过努力，早在 2014 年，就有「微软开发交互式动态磁贴」的消息流出。从当时流出的视频来看，微软将部分承载大量信息的动态磁贴放大，并赋予其滑动、下级菜单等交互功能，类似安卓系统中的「小部件」。

从 2014 年开始，几乎每年都会有「交互式动态磁贴」的新消息出现。

▲ 可以滑动的动态磁贴

但直到 2016 年，这些功能仍处在研究阶段。当时安卓和 iOS 已经出现了信息聚合页，将大量的信息以卡片的形式聚合在一个专门的页面上。后台控制则放进下拉通知栏中，真正有用的小部件也只剩天气展示和备忘录展示。大家更喜欢信息流，而不是杂乱的磁贴。

这时的 PC ，已经进入了新时代，固态的普及让大部分应用可以实现秒开。在应用可以秒开的情况下，人们更愿意打开应用全屏浏览信息，而不是在一个小方块中点来点去。

在这些因素的综合影响下，动态磁贴的存在感越来越弱。到了 Windows 10 的时代，传统开始菜单重新回归，这时的动态磁贴几乎只剩下展示天气的功能。除了寥寥几个系统内置应用外，很少有第三方应用适配动态磁贴功能，动态磁贴实际上已经名存实亡。

微软也终于放弃了挣扎，连续好几次年度更新，都没有再为动态磁贴添加新的功能。直到 2017 年 Fluent Design 的公布，基本上宣布了「交互式动态磁贴」计划的流产。相比继续在动态磁贴上折腾，微软更愿意提高系统的 UI 设计，用外观来吸引用户。

▲ Fluent Design

动态磁贴还有未来么？

动态磁贴之所以没落，根本原因还是卡片形态对文字信息流没有足够的承载力。在信息爆炸时代，时间轴才是更好的信息集散地。

好在磁贴除了承载信息，还有其他的意义。相对普通的桌面图标，可变排布的磁贴有着更大的自定义空间。

▲ 图片来自：知乎

这些磁贴支持不同的大小和形状，对用户来说，它就是拥有无限可能的乐高积木。只要你的想象力足够强大，这些磁贴可以有无限种组合的可能。

比如用磁贴拼出图案，或者用大小颜色不同的磁贴标注出高频应用，提高应用的打开效率。或者用将磁贴做成海报墙，让自己的桌面更加个性。

▲ 磁贴自定义图片来自：知乎

但这些都不是磁贴的终极形态，磁贴最终的归宿，必然是承载信息。但这些信息不会是简单的文字，而是经过 AI 和算法分析归纳后的数据，这些图形化、可交互的数据，将成为信息与人类间的新桥梁。

▲ 或许动态磁贴将以归纳图表的形态重生 图片来自：allegro

再见，动态磁贴。
