去年年底，vivo 用 NEX 3 开启了瀑布屏的时代，在将手机两侧的中框削减到极限的同时，首次用压感按键全面替换实体按键，推动了手机形态再次进化。同时，NEX 3 也是 vivo 高端系列中第一款支持 5G 的旗舰手机。

半年过去，高通新一代 X55 基带已经上市，新的 SoC 能带给更好的 5G 体验。NEX 系列作为 vivo 高端旗舰的排面，当然不能落于人后，所以 vivo 对 NEX 3 进行了升级，它就今天发布会的主角—— NEX 3S。

2020 年 3 月 10 日，vivo 在线上举办了 NEX 3S 的线上发布会。发布会一开始，vivo NEX 系列的产品经理就宣布 vivo 推出了无接触购机服务，并对疫情期间保修过期的手机进行延保。

随后 vivo NEX 3S 正式登场，作为 NEX 3 的升级，vivo NEX 3S 当然配备了今年旗舰的高通全新骁龙 865 + X55 芯片，支持 NSA/SA 双模 5G 。除此以外，NEX 3S 还全系标配 UFS 3.1 存储芯片和最新的 LPDDR5 内存，并配上 Multi-Turbo 3.0 和 VPG 系统进程监控技术，保证旗舰性能完美发挥。

与 NEX 3 一样，NEX 3S 同样采用了 4500 mAh 的电池，继续使用 Super FlashCharge 44W 超级闪充。vivo NEX 3S 搭载了零感均热板散热，相比常见的液冷热管散热面积更大、热阻更低，散热效果更显著，机身平均温度能下降 3~4 摄氏度。

除了 5G 和 SoC 性能的进一步进化外，NEX 3S 同样支持 WiFi 6，并支持 WiFi 双频段连接，可以同时连接两个 2.4G/5G 两个频段。NEX 3S 同时采用了 BT High-power 技术，通过提升蓝牙功率，让蓝牙穿墙效果进一步得到提升。

除了与网络连接之外，与其他手机的分享也非常重要，NEX 3S 对 Android Beam NFC 互传进行了改进，通过 Wi-Fi P2P 直连技术和虚拟技术，实现了手机应用的快捷投屏分享，只要轻轻一碰，手机内的内容就会以小窗口的形态传输到另一台手机中。

屏幕方面，NEX 3S 继承了 NEX 3 的屏幕配置，搭载 6.89 英寸无界瀑布屏，左右两侧弯折接近 90°，屏占比达到 99.6% ，曲面边缘能够有效防止误触。

屏幕素质方面，NEX 3S 屏幕支持 100% P3 色域覆盖，局部最高亮度 800nit ，屏幕材质方面继续采用更加护眼的 E3 材质，并支持 HDR10 和 HDR10+ 视频播放。

由于瀑布屏挤占中框，所以 NEX 3S 继承了 NEX 3 取消实体按键采用虚拟压感按键的设计，至于按键反馈，则用 X 轴线性马达来完成。

屏幕占用中框还带来了另一种好处，NEX 3S 的曲面屏支持边缘触控操作，可以实现类似侧边按键的功能。X 轴线性马达则可以接近真实的 4D 游戏震感。vivo NEX 3S 还带来了全新的模拟机械键盘，能通过震动马达模拟机械键盘的双段式触感。

配色方面，NEX 3S 新增了琥珀醇配色，灵感由天然琥珀而来。加上液态天河、深空流光两种配色，总共有 3 种配色可选。

系统方面，vivo NEX 3S 重绘了桌面图标和物理动效 ，并推出了一款新的浏览器—— NEX 极简浏览器，去掉了繁杂的功能与信息，搭配过滤广告功能，为用户提供极致的浏览服务。

相机方面，NEX 3S 继承了 NEX 3 的相机配置，搭载了 64M 像素 1/1.7 英寸 CMOS ，13M 广角微距 + 13M 长焦镜头。对比 NEX 3 ，在人像模式中加入了人眼追焦功能。

视频拍摄方面，NEX 3S 在主摄像头和超广角摄像头上同时搭载了超级 EIS 视频防抖，防抖效果显著提升的同时，防抖后的画面视角也进一步扩大。

其他方面，NEX 3S 和 NEX 3 都是一样的，同样支持 Hi-Fi 和 3.5mm 耳机接口。

甚至售价策略也与 NEX 3 相同，vivo NEX 3S 开售价格和 NEX 3 4G 版相同，vivo NEX 3S 8GB + 256GB 售价 4998 元；12GB + 256GB 版本售价 5298 元。你只要花去年 1 部 4G 版 NEX 3 的价格，就能买到一部 5G 版、还是骁龙 865 处理器的 NEX 3S。对于喜欢瀑布屏设计的人，NEX 3S 可谓理想的选择。

加量不加价，说的就是 NEX 3S 。
